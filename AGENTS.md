<!-- convex-ai-start -->

This project uses [Convex](https://convex.dev) as its backend.

Convex functions use string paths (e.g. `'cars:list'`) in the frontend.
The `convex/` directory must stay at the project root (Convex CLI requirement).

Convex agent skills for common tasks can be installed by running
`npx convex ai-files install`.
